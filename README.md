# Jumping_Frog
This game was created for my Basics of Computer Programming class. It is my first project for said class. It's supposed to mimic the game "crossy road", but extremely simplified and not too complicated. 


## Compiling

### Windows
Good luck 👍

### Linux/Mac:
- _Open terminal_
- _Install the library ncurses (lncurses, curses)_
- _Download files to computer and store in one folder_
- _In said folder, type:_
- *make*
- *make run*
- *make clean*
